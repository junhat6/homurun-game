# Task Completion Checklist

タスク完了時に確認すること：

## 1. コード品質
- [ ] コンソールエラーがないか確認
- [ ] 不要なconsole.logを削除
- [ ] コードが読みやすく整理されているか

## 2. 動作確認
- [ ] ブラウザでゲームが正常に動作するか
- [ ] スペースキーでバットが振れるか
- [ ] クリック/タッチでバットが振れるか
- [ ] スコアが正しく加算されるか

## 3. レスポンシブ対応（任意）
- [ ] 異なる画面サイズで表示確認
- [ ] モバイルでタッチ操作が動作するか

## 4. Git/デプロイ
```bash
# 変更をコミット
git add .
git commit -m "変更内容を記述"
git push origin main
```

## 5. GitHub Pages確認
- 公開URL: https://junhat6.github.io/homurun-game/
- デプロイ後、数分待ってからアクセス確認

## 6. 提出前最終確認
- [ ] URLが正しい形式か (`https://USERNAME.github.io/REPO/`)
- [ ] URLをクリックしてゲームがすぐ開くか
- [ ] 公序良俗に反する内容がないか
