## Game Concept
スマブラのホームランコンテスト風ゲーム。

### ゲームフロー
1. 制限時間10秒でサンドバッグ（坪内先生）を殴ってダメージを溜める
2. スペースキーでバットスイング（吹っ飛ばし）
3. Planck.js物理エンジンでリアルな放物線
4. 飛距離を計測

### 操作方法
- `A` / `D` : 左右移動
- `W` : ジャンプ（2段ジャンプ可能）
- `J` : 弱攻撃（速い、コンボ始動）
- `K` : 強攻撃（中ダメージ）
- `L` : スマッシュ攻撃（長押しで溜め可能、高ダメージ）
- `SPACE` : バットスイング（最終打撃、吹っ飛ばし）

### ゲームシステム
- **コンボ**: 連続ヒットでダメージボーナス
- **ノックバック**: ダメージ%に応じてノックバック増加（スマブラ式）
- **スマッシュ溜め**: Lキー長押しで威力アップ
- **ヒットストップ**: 攻撃ヒット時のフィードバック

### Scoring
- ダメージ％が高いほど飛距離が伸びる
- 最終飛距離(m)がスコア
